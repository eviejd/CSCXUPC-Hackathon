How To Use

1. Add users by entering their names and clicking "Add User".

2. Add tasks to the queue by entering task descriptions and clicking "Add Task".

3. Once you have at least one user and one task, the "Start Round" button will be enabled.

4. Click "Start Round" to begin the bidding process.

5. Recieve your set tasks once bidding concludes.



What happens if we run out of Money?

Each user starts with a set amount of points (e.g., 100). Bids are deducted from their total points.

If a user runs out of points, they can no longer place bids.

This now means you can not longer get out of tasks.

While losers' tasks is less than average tasks, losers will be assigned tasks other member want to bid on, to not get assigned.
